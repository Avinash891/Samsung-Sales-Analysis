
# Samsung Global Sales Analysis Dashboard

##  Project Overview
An interactive Power BI dashboard analyzing 15,500+ sales transactions across 4 years (2021-2024) to provide actionable insights into global sales performance, customer behavior, and market trends.

##  Objectives
- Track key business metrics: revenue, units sold, customer ratings, and regional performance
- Identify top-performing markets and product categories
- Analyze seasonal demand patterns and payment preferences
- Enable data-driven decision-making through interactive visualizations

##  Key Metrics
- **Total Revenue:** $22.91M
- **Units Sold:** 37K
- **Total Transactions:** 15,500+
- **Average Customer Rating:** 3.73/5
- **Time Period:** 2021-2024
- **Geographic Coverage:** Multiple countries including Argentina, USA, and more

##  Tools & Technologies
- **Power BI:** Dashboard development and visualization
- **Power Query:** Data cleaning and transformation
- **DAX:** Advanced calculations and measures
- **Excel:** Initial data processing

##  Key Features
- **Geographic Analysis:** Heat maps showing sales distribution across countries
- **Trend Analysis:** Monthly and yearly revenue patterns
- **Category Performance:** Product category breakdown with sales insights
- **Payment Analytics:** Payment method preferences by region
- **Customer Segmentation:** Analysis by demographics and purchase behavior
- **Interactive Filters:** Dynamic filtering by year, category, and country

##  Key Insights
1. **Top-performing regions** contribute 40% of total revenue
2. **Seasonal patterns** show peak sales in specific months
3. **Payment method preferences** vary significantly by geography
4. **5G products** show different adoption rates across markets
5. **Customer age groups** display distinct purchasing behaviors

##  Dataset Information
- **Records:** 15,500+ transactions
- **Columns:** 28 attributes
- **Features:** Sales date, product details, customer demographics, payment info, geographic data, ratings

##  How to Use
1. Download the `.pbix` file from this repository
2. Open in Power BI Desktop
3. Explore interactive visualizations using slicers and filters
4. Dataset included in the repository for reference
